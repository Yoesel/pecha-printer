# Pecha Printer
Standalone app converting a pdf into a ready to print "Pecha", traditional tibetan volumes.

## Install
### Version 01

* Download [Windows 64 installer](https://github.com/Esukhia/dakje/releases/download/beta_01/Dakje_beta_01.exe)
